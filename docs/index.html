Adding people to the "discuss" listserv

Anybody can subscribe themselves to the discuss listserv, but only admins can add others to discuss.

  • go to the somervillebikekitchen.org admin console:
    https://admin.google.com/somervillebikekitchen.org/AdminHome
  • login as "admin"
  • click on Groups
  • click on the discuss listserv
  • click on Manage users in discuss
  • paste the email addresses in the text box (comma delimited)
  • select Member from the drop-down and click Add

Use this same procedure for the info and announce listservs.

Editing the website

Prerequisites: git, make.
  • clone the website using git:
    git clone https://github.com/somervillebikekitchen/somervillebikekitchen.github.io.git
    or if you've already cloned before, make sure you have the latest version:
    cd path/to/sbk
    git pull
  • make your desired changes
  • spin up test server:
    make run
  • navigate to http://localhost:8000/ in your web browser
  • make sure your changes look correct
  • commit your changes:
    git commit -am "Your reason for change"
  • create a patch file with your changes:
    git format-patch origin/master
    this will generate a patch file and give you a filename; attach this file to an email and send it to info _at_ somervillebikekitchen (dot) org. Alternatively, if you have a github account and you've been added to the repository as a collaborator, you can just push and instantly update the website:
    git push
