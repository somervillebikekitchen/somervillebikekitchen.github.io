adding people to the "discuss" listserv

  • go to the somervillebikekitchen.org admin console:
    https://admin.google.com/somervillebikekitchen.org/AdminHome
  • login using your somervillebikekitchen.org google apps credentials
  • click on Groups
  • click on the discuss listserv
  • click on Manage users in discuss
  • paste the email addresses in the text box (comma delimited)
  • select Member from the drop-down and click Add

open shop scheduling

  • we keep a calendar in the shop

core staff onboarding

  • verify that the new staff member has met all the onboarding criteria
  • add the new staff to the discuss mailing list using the instructions above
  • record the new staff member's information in a physical list inside the shop
    • full name
    • email address
    • alternative contact

editing the website

  • clone the website using git:
    git clone https://github.com/somervillebikekitchen/somervillebikekitchen.github.io.git
    or
    git clone http://troy.sankey.info/repos/sbk.git
  • make your desired changes
  • spin up test server:
    make run
  • navigate to http://localhost:8000/ in your web browser
  • make sure your changes look correct
  • commit your changes:
    git commit -am "Your reason for change"
  • create a patch file with your changes:
    git format-patch origin/master
    this will generate a file and give you a filename; attach this file to an email and send it to info _at_ somervillebikekitchen (dot) org.
